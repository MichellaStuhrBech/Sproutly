# Advertising assets

The dashboard ad card shows the **TomatoSale** image.

**To display your picture:** put your image file at  
`frontend/public/Advertising/TomatoSale.png`  
(or `TomatoSale.jpg` and change the image path in `DashboardPage.jsx` to `/Advertising/TomatoSale.jpg`).

You can keep a copy in this folder (`src/Advertising/`) for reference; the app loads the image from `public/Advertising/`.
